# Cisco Packet Tracer Labs

This repository contains **8 Cisco Packet Tracer labs** created to practice and understand basic networking concepts.

## Topics Covered
- Network topologies
- IP addressing and subnetting
- Static and dynamic routing
- VLANs and inter-VLAN routing
- Switching basics
- Connectivity troubleshooting

## Tool Used
- Cisco Packet Tracer
